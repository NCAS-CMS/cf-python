cf.Field.Flags¶

Field.Flags¶

A Flags object containing self-describing CF flag values.

Stores the flag_values, flag_meanings and flag_masks CF properties in an internally consistent manner.

Examples:

>>> f.Flags
<CF Flags: flag_values=[0 1 2], flag_masks=[0 2 2], flag_meanings=['low' 'medium' 'high']>
