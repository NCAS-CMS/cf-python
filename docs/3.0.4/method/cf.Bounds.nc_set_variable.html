cf.Bounds.nc_set_variable¶

Bounds.nc_set_variable(value)[source]¶

Set the netCDF variable name.

New in version 1.7.0.

See also

nc_del_variable, nc_get_variable, nc_has_variable

Parameters:
value: str

The value for the netCDF variable name.
Returns:

None

Examples:

>>> f.nc_set_variable('tas')
>>> f.nc_has_variable()
True
>>> f.nc_get_variable()
'tas'
>>> f.nc_del_variable()
'tas'
>>> f.nc_has_variable()
False
>>> print(f.nc_get_variable(None))
None
>>> print(f.nc_del_variable(None))
None
