cf.Count.all¶

Count.all()[source]¶

Test whether all data elements evaluate to True.

Performs a logical “and” over the data array and returns the result. Masked values are considered as True during computation.

See also

allclose, any

Returns
bool

Whether ot not all data elements evaluate to True.

Examples:

>>> print(f.array)
[[0  3  0]]
>>> f.all()
False

>>> print(f.array)
[[1  3  --]]
>>> f.all()
True
