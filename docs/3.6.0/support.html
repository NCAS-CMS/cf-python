Support¶

Version 3.6.0 for version 1.8 of the CF conventions.

General questions, suggestions for enhancements, and reports of bugs may be reported in the GitHub issue tracker: https://github.com/NCAS-CMS/cf-python/issues

Contributing¶

Contributions to cf-python are welcome and should be raised as issues in the GitHub issue tracker: https://github.com/NCAS-CMS/cf-python/issues, prior to submitting a pull request containing the new code made from a fork of the cf-python GitHub code repository: https://github.com/NCAS-CMS/cf-python.
