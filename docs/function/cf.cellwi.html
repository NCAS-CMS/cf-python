cf.cellwi¶

cf.cellwi(value0, value1, units=None)[source]¶

A Query object for a “cell bounds lie within range” condition.

See also

cf.cellsize, cf.contains, cf.cellge, cf.cellgt, cf.cellne, cf.cellle, cf.celllt, cf.cellwo

Parameters
value:

The query condition’s value.

units: str or Units, optional

The units of value. By default, the same units as the operand being tested are assumed, if applicable. If units is specified and the already value has units (such as a Data object`), then they must be equivalent.

Returns
Query

The query object.
