cf.CoordinateConversion.clear_parameters¶

CoordinateConversion.clear_parameters()[source]¶

Remove all parameters.

New in version 1.7.0.

See also

del_parameter, parameters, set_parameters

Returns
dict

The parameters that have been removed.

Examples:

>>> old = f.clear_parameters()
>>> old
{'standard_parallel': 25.0;
 'longitude_of_central_meridian': 265.0,
 'latitude_of_projection_origin': 25.0}
>>> f.parameters()
{}
>>> f.set_parameters(old)
>>> f.parameters()
{'standard_parallel': 25.0;
 'longitude_of_central_meridian': 265.0,
 'latitude_of_projection_origin': 25.0}
