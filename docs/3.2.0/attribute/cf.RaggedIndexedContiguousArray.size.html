cf.RaggedIndexedContiguousArray.size¶

RaggedIndexedContiguousArray.size¶

Number of elements in the uncompressed data.

Examples:

>>> d.shape
(73, 96)
>>> d.size
7008
>>> d.ndim
2

>>> d.shape
(1, 1, 1)
>>> d.ndim
3
>>> d.size
1

>>> d.shape
()
>>> d.ndim
0
>>> d.size
1
