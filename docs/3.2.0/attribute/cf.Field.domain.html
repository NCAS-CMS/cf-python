Field.domain¶

Return the domain.

f.domain is equivalent to f.get_domain()

New in version 1.7.0.

See also

get_domain

Returns
Domain

The domain.

Examples:

>>> d0 = f.domain
>>> d1 = f.get_domain()
>>> d0.equals(d1)
