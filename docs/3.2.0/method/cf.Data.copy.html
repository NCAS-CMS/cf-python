cf.Data.copy¶

Data.copy(array=True)[source]¶

Return a deep copy.

d.copy() is equivalent to copy.deepcopy(d).

Parameters
array: bool, optional

If False then do not copy the array. By default the array is copied.

Returns

The deep copy.

Examples:

>>> e = d.copy()
>>> e = d.copy(array=False)
