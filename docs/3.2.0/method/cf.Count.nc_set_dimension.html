cf.Count.nc_set_dimension¶

Count.nc_set_dimension(value)[source]¶

Set the netCDF dimension name.

New in version 1.7.0.

See also

nc_del_dimension, nc_get_dimension, nc_has_dimension

Parameters
value: str

The value for the netCDF dimension name.

Returns

None

Examples:

>>> f.nc_set_dimension('time')
>>> f.nc_has_dimension()
True
>>> f.nc_get_dimension()
'time'
>>> f.nc_del_dimension()
'time'
>>> f.nc_has_dimension()
False
>>> print(f.nc_get_dimension(None))
None
>>> print(f.nc_del_dimension(None))
None
