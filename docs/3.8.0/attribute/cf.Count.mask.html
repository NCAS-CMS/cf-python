cf.Count.mask¶

Count.mask¶

The mask of the data array.

Values of True indicate masked elements.

See also

binary_mask

Examples:

>>> f.shape
(12, 73, 96)
>>> m = f.mask
>>> m.long_name
'mask'
>>> m.shape
(12, 73, 96)
>>> m.dtype
dtype('bool')
>>> m.data
<CF Data(12, 73, 96): [[[True, ..., False]]] >
