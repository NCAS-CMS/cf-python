cf.List.ndim¶

List.ndim¶

The number of dimensions in the data array.

See also

data, has_data, isscalar, shape, size

Examples:

>>> f.shape
(73, 96)
>>> f.ndim
2
>>> f.size
7008

>>> f.shape
(73, 1, 96)
>>> f.ndim
3
>>> f.size
7008

>>> f.shape
(73,)
>>> f.ndim
1
>>> f.size
73

>>> f.shape
()
>>> f.ndim
0
>>> f.size
1
