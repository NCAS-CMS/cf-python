cf.DimensionCoordinate.decreasing¶

DimensionCoordinate.decreasing()¶

True if the dimension coordinate is increasing, otherwise False.

A dimension coordinate is increasing if its coordinate values are increasing in index space.

The direction is inferred from one of, in order of precedence:

  • The data array
  • The bounds data array
  • The units CF property
Returns:
out : bool

Whether or not the coordinate is increasing.

True for dimension coordinate constructs, False otherwise.

>>> c.decreasing
False
>>> c.flip().increasing
True
