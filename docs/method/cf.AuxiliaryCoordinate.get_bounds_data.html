cf.AuxiliaryCoordinate.get_bounds_data¶

AuxiliaryCoordinate.get_bounds_data(default=ValueError())[source]¶

Return the bounds data.

New in version 1.7.0.

See also

bounds, get_bounds, get_data

Parameters
default: optional

Return the value of the default parameter if there are no bounds data. If set to an Exception instance then it will be raised instead.

Returns

The bounds data.

Examples:

>>> c.get_bounds_data()
<Data(96, 2): [[0, ..., 360.0]] degrees_east>
