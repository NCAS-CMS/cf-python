cf.AuxiliaryCoordinate.bounds¶

AuxiliaryCoordinate.bounds()¶

Return the bounds.

f.bounds is equivalent to f.get_bounds()

New in version 1.7.0.

See also

data, del_bounds, get_bounds, has_bounds, set_bounds

Returns:
Bounds

The bounds.

Examples:

>>> import numpy
>>> b = cfdm.Bounds(data=cfdm.Data(numpy.arange(10).reshape(5, 2)))
>>> c.set_bounds(b)
>>> c.has_bounds()
True
>>> b = c.bounds
>>> b
<Bounds: (5, 2) >
>>> b.data
<Data(5, 2): [[0, ..., 9]]>
>>> b.data.shape
(5, 2)
