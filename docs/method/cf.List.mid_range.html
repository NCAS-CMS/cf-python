cf.List.mid_range¶

List.mid_range()[source]¶

The unweighted average of the maximum and minimum of the data array.

See also

maximum, mean, minimum, range, sample_size, sd, sum, var

Returns
Data

The unweighted average of the maximum and minimum of the data array.

Examples:

>>> f.data
<CF Data(12, 73, 96): [[[236.512756348, ..., 256.93371582]]] K>
>>> f.mid_range()
