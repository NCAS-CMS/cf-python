cf.CellMethod.set_axes¶

CellMethod.set_axes(value, copy=True)[source]¶

Set the axes of the cell method.

New in version 1.7.0.

See also

del_axes, get_axes, has_axes

Parameters:
value: (sequence of) str

The axes, specified either by the construct identifiers of domain axis constructs, standard names, or the special string 'area'.

Parameter example:

axes='domainaxis0'

Parameter example:

axes='time'

Parameter example:

axes='area'

Parameter example:

axes=['domainaxis0', 'domainaxis2']

Parameter example:

axes=['time', 'area']

Parameter example:

axes=['domainaxis0', 'time']
Returns:

None

Examples:

>>> c.set_axes('domainaxis1')
>>> c.has_axes()
True
>>> c.get_axes()
('domainaxis1',)
>>> c.del_axes()
>>> c.has_axes()
False
>>> c.get_axes('NO AXES')
'NO AXES'
>>> c.del_axes('NO AXES')
'NO AXES'

>>> c.set_axes(['domainaxis1', 'domainaxis0'])

>>> c.set_axes(['time', 'domainaxis0'])

>>> c.set_axes('time')
