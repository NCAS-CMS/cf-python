cf.Bounds.ctype¶

Bounds.ctype¶

The CF coordinate type.

One of 'T', 'X', 'Y' or 'Z' if the coordinate construct is for the respective CF axis type, otherwise None.

See also

T, X, Y, Z

Examples:

>>> c.X
True
>>> c.ctype
'X'

>>> c.T
True
>>> c.ctype
'T'
