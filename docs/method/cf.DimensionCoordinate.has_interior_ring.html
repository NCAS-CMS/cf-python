cf.DimensionCoordinate.has_interior_ring¶

DimensionCoordinate.has_interior_ring()[source]¶

Whether or not there is an interior ring variable.

New in version (cfdm): 1.8.0

See also

del_interior_ring, get_interior_ring, interior_ring, set_interior_ring

Returns
bool

True if there is an interior ring variable, otherwise False.

Examples:

>>> import numpy
>>> c = cf.DimensionCoordinate()
>>> i = cf.InteriorRing(data=cf.Data(numpy.arange(10).reshape(5, 2)))
>>> c.set_interior_ring(i)
>>> c.has_interior_ring()
True
>>> i = c.get_interior_ring()
>>> i
<CF InteriorRing: (5, 2) >
>>> i.data
<CF Data(5, 2): [[0, ..., 9]]>
>>> i.data.shape
(5, 2)
>>> c.del_interior_ring()
<CF InteriorRing: (5, 2) >
>>> c.has_interior_ring()
False
>>> print(c.del_interior_ring(None))
None
