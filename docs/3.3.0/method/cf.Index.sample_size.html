cf.Index.sample_size¶

Index.sample_size()[source]¶

The number of non-missing data elements in the data array.

See also

count, maximum, mean, mid_range, minimum, range, standard_deviation, sum, variance

Returns
Data

The number of non-missing data elements in the data array.

Examples:

>>> f.data
<CF Data(12, 73, 96): [[[236.512756348, ..., 256.93371582]]] K>
>>> f.sample_size()
<CF Data(): 98304.0>
